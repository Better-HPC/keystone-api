# API Specification

All Keystone API releases include a complete OpenAPI 3.0 specification in both YAML and JSON formats.
These documents provide a complete, version-specific description of the API, including available endpoints,
supported parameters, request body schemas, response payloads, and authentication flows.

A rendered version of the latest OpenAPI specification is available [in the online documentation](swagger.html).
API specifications for specific API versions can be fetched from the API itself at the following endpoints.

| Endpoint         | Description                              |
|------------------|------------------------------------------|
| `/openapi/yaml/` | OpenAPI 3.0 specification in YAML format |
| `/openapi/json/` | OpenAPI 3.0 specification in JSON format |
